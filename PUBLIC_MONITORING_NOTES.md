# Public Route Notes for Octra HFHE v2

## Scope

This document tracks public signals for Octra HFHE Challenge v2 and summarizes routes that have not produced a verified recovery.

## Excluded from this repository

- GitHub tokens and github.txt
- local wallet/private secrets
- challenge ciphertext/public-key binaries
- unpublished key material
- bulk workspace logs outside the whitelist

## Routes watched

1. Chain-state changes for the target wallet.
2. Official repository updates, forks, PRs, and issues.
3. Public X/search mentions for solved claims or leaked artifacts.
4. Specific target materials: Rku, rk.sec, NatKey, eval-key/recrypt bundles.
5. LPN/sample-based routes.

## Current result

No verified plaintext or private key has been recovered. Latest machine-readable state is summarized in STATUS_LATEST.md.
